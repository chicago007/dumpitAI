# dumpitAI 사용설명서

> 생각을 실행으로 바꿔주는 AI 비서

dumpitAI는 머릿속에 떠오른 일을 한 줄로 적으면 **할일 · 일정 · 메모**로 자동 분류하고, 여행·출장 같은 큰 일은 **프로젝트**로 묶어 체크리스트·예산·지출까지 한곳에서 관리하는 앱입니다.

## 1. 시작하기

1. 앱에 접속하면 로그인 화면이 나옵니다.
2. 계정이 없으면 **회원가입**으로 이메일·비밀번호를 등록합니다.
3. 로그인하면 바로 **입력** 화면으로 이동합니다.

> 처음 로그인 시 기본 카테고리가 자동으로 만들어집니다.

---

## 2. 화면 구성

왼쪽(모바일은 좌측 상단 메뉴 버튼)의 사이드바로 이동합니다.

| 메뉴 | 설명 |
|------|------|
| **입력** | 한 줄/여러 줄을 적어 빠르게 저장하는 메인 화면 |
| **오늘** | 오늘 마감인 할 일·일정과 오늘 완료한 항목 |
| **달력** | 월간 달력에서 일정 확인·추가 |
| **메모** | 저장한 메모 목록 |
| **할일** | 할 일 목록 |
| **일정** | 일정 목록 |
| **프로젝트** | 여행·출장 등 묶음 관리 |
| **완료** | 완료한 항목과 이번 주 통계 |
| **설정** | 테마·기본 설정 |

사이드바 각 메뉴 옆 숫자는 해당 항목의 개수입니다.

## 3. 개인 / 업무 공간

사이드바 상단의 **공간 전환기**로 보기를 바꿀 수 있습니다.

- **개인** — 여행·생활·운동 등 개인 항목
- **업무** — 회의·보고서·프로젝트 등 업무 항목
- **전체** — 개인과 업무를 한꺼번에 표시

선택한 공간은 모든 목록·달력·통계에 반영되고, 다음 방문에도 유지됩니다.

### 전체 보기에서 저장할 때

전체 보기 상태로 입력하면 앱이 내용으로 공간을 추정합니다. 정확히 지정하려면 줄 맨 앞에 접두어를 붙입니다.

- `/업무 분기 보고서 작성` → 업무로 저장
- `/개인 주말 등산 준비물` → 개인으로 저장

(`/업`, `/개`로 줄여 써도 됩니다.)

---

## 4. 빠른 입력

**입력** 화면의 입력창에 적고 Enter를 누르면 저장됩니다.

- **한 줄 입력**: 내용에 맞춰 할일·일정·메모로 자동 분류됩니다.
  - 날짜가 있으면 → **일정** (예: `금요일 3시 치과`)
  - "~까지"가 있으면 → **할일** (예: `금요일까지 보고서 제출`)
  - "회의·약속·미팅" 등 → **일정**, "제출·준비·구매" 등 → **할일**
- **종류를 직접 지정**하려면 `@` 접두어를 사용합니다.
  - `@메모 비밀번호 힌트`
  - `@할일 우유 사기`
  - `@일정 내일 2시 미팅`

### 날짜·시간 표현

다음과 같은 자연어 날짜를 인식합니다.

- `오늘`, `내일`, `모레`
- `월요일`, `화요일` … `일요일`
- `8월 3일`, `8/3`, `3시`, `오후 2시`

## 5. AI 분류 입력

여러 줄을 한꺼번에 붙여넣으면 AI가 줄마다 할일·일정·메모로 나누고, 미리보기에서 **저장 전 수정**할 수 있습니다.

예시:

```
다음 주 수요일 팀 미팅
금요일까지 디자인 시안 보내기
회식 장소 후보 메모
```

- 각 줄의 **분류**(할일/일정/메모)와 **공간**(개인/업무)을 드롭다운으로 바꿀 수 있습니다.
- 첫 줄을 `프로젝트 제주여행` 또는 `@p 제주여행`으로 시작하면, 나머지 줄을 그 **프로젝트**로 묶어 저장합니다.
- "○○여행"처럼 여행 프로젝트가 감지되면 자동으로 프로젝트로 제안됩니다.

---

## 6. 목록 페이지 (할일 · 일정 · 메모)

각 목록에서 항목을 관리합니다.

- **완료 체크**: 왼쪽 체크박스를 누르면 완료 처리됩니다.
- **수정**: 항목 위에 마우스를 올리면(모바일은 항상 표시) 연필 아이콘으로 내용·종류·카테고리·날짜를 수정합니다.
- **삭제**: 휴지통 아이콘으로 삭제합니다.
- **공간 이동**: 수정 화면에서 개인 ↔ 업무로 옮길 수 있습니다.

색상 막대는 종류를 나타냅니다 — 할일(파랑), 일정(보라), 메모(주황), 체크리스트(초록).

---

## 7. 오늘 · 달력 · 완료

### 오늘
- 오늘 마감인 할 일과 일정을 모아 보여줍니다.
- 아래쪽 **오늘 완료** 영역에서 오늘 끝낸 항목을 개인/업무·종류별로 확인합니다.

### 달력
- 월간 달력에서 일정을 한눈에 봅니다.
- 날짜를 선택해 그날의 일정을 보거나 새 일정을 추가합니다.
- 프로젝트에 속한 일정은 전체 달력과 분리되어, 프로젝트의 **일정 탭**에서만 관리됩니다.

### 완료
- 완료한 항목을 **개인/업무 + 종류**로 묶어 보여줍니다(전체 보기 시).
- 상단에 이번 주 완료 건수가 표시됩니다.

## 8. 프로젝트

여행·출장·공부처럼 여러 할일·일정·예산이 얽힌 큰 일을 하나로 묶어 관리합니다.

### 프로젝트 만들기
- **프로젝트** 화면에서 새로 만들거나, AI 입력에서 `프로젝트 이름`으로 시작해 만듭니다.
- 유형(여행·출장·캠핑·공부·업무·직접입력)에 따라 체크리스트·예산 카테고리가 자동으로 채워집니다.

### 프로젝트 기간
- `8월 1일 ~ 10일` 형식으로 전체 기간을 지정하면 프로젝트 이름 옆에 표시됩니다.
- 목록·상세 화면에서 기간을 수정할 수 있습니다.

### 프로젝트 탭

| 탭 | 설명 |
|------|------|
| **체크리스트** | 준비물·할일을 카테고리별로 관리. 진행률 표시 |
| **일정** | 프로젝트 전용 일정(출국·체크인 등). 전체 달력과 분리 |
| **메모** | 프로젝트 관련 메모 |
| **예산** | 총 예산과 카테고리별 예산. 사용/잔액 자동 계산 |
| **지출** | 실제 지출 기록. 통화별 입력 가능 |
| **AI** | AI 추천 준비물 |

### 프로젝트 안에서 빠르게 입력하기

프로젝트 상세의 입력창은 내용에 따라 자동 분류되며, `@` 접두어로 지정할 수도 있습니다.

- `@기간 8월 1일 ~ 10일` → 프로젝트 기간
- `@일정 8월 1일 출국` → 일정
- `@예산 총예산 200만원` → 예산
- `@지출 점심 12,000원` → 지출
- `@메모 호텔 체크인 15시` → 메모
- `기본/여권` → '기본' 카테고리에 '여권' 체크리스트 추가
- `숙박 호텔 30만원` → 금액이 붙은 체크리스트 항목

### 지출 관리
- **지출 추가**: 금액·통화·카테고리·메모를 입력해 기록합니다(원/달러/엔/유로 지원).
- **지출 카테고리**: 지출 추가 영역의 "지출 카테고리"를 펼쳐 카테고리를 추가·수정·삭제합니다. (이름을 바꾸면 기존 지출도 함께 바뀌고, 삭제 시 해당 지출은 '기타'로 이동)

## 9. 설정

**설정** 화면에서 화면 테마를 고를 수 있습니다.

| 테마 | 설명 |
|------|------|
| 기본 | 밝은 회색 배경 |
| 흰색 | 깔끔한 흰 배경 |
| 따뜻한 | 아이보리 톤 |
| Solarized Dark | Solarized 다크 팔레트 |
| 미드나잇 | 진한 슬레이트(다크) |

선택한 테마는 기기에 저장되어 다음 방문에도 적용됩니다.

## 10. 입력 문법 요약

### 종류 지정 (`@`)

| 입력 | 결과 |
|------|------|
| `@할일` / `@t` | 할일 |
| `@일정` / `@s` | 일정 |
| `@메모` / `@m` | 메모 |
| `@체크리스트` | 체크리스트(프로젝트) |
| `@예산` | 예산(프로젝트) |
| `@지출` | 지출(프로젝트) |
| `@기간` | 프로젝트 기간 |
| `@p 이름` / `프로젝트 이름` | 프로젝트 블록 시작 |

### 공간 지정 (`/`)

| 입력 | 결과 |
|------|------|
| `/업무` 또는 `/업` | 업무로 저장 |
| `/개인` 또는 `/개` | 개인으로 저장 |

### 자동 분류 규칙

- 날짜만 있으면 → **일정**
- "~까지"가 있으면 → **할일**
- 금액 + 준비/예약 키워드 → 프로젝트 **체크리스트**
- 금액 + 식비/결제 키워드 → 프로젝트 **지출**
- 길거나 목록형(- 로 시작) → **메모**

## 11. 자주 묻는 질문

**Q. 한 줄을 적었는데 종류가 잘못 분류돼요.**
저장 후 항목 수정에서 종류를 바꾸거나, 입력 시 `@할일`·`@일정`·`@메모` 접두어로 지정하세요.

**Q. 전체 보기에서 저장하니 공간을 물어봐요.**
전체 보기에서는 개인/업무를 추정합니다. `/업무`·`/개인`을 줄 앞에 붙이면 바로 저장됩니다.

**Q. 완료에서 개인 항목이 안 보여요.**
공간 전환기를 **전체** 또는 **개인**으로 두고 확인하세요. 카테고리 기준으로도 개인/업무를 구분합니다.

**Q. 프로젝트 일정이 달력에 안 나와요.**
의도된 동작입니다. 프로젝트 일정은 해당 프로젝트의 **일정 탭**에서 관리됩니다.

**Q. 화면이 어두운 테마에서 일부 요소가 흰색으로 보여요.**
설정에서 다크 테마를 선택하면 입력칸·드롭다운까지 테마에 맞춰집니다. 최신 버전으로 새로고침해 보세요.
